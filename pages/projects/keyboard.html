Keyboard Party

UnityC#Networking

During my studies in the design course, I created a game that relied on a unique input method: using the entire keyboard, utilizing over 30 keys. This project served as an opportunity for me to delve into how to develop an online game in Unity using Unity Gaming Services, specifically focusing on its replay system.

In addition to the online system, the game features a single-player mode with several minigames. I worked on this project with my partner, Enrique Tarín.

In this project, I undertook the following tasks:
- Implementation of the online system and online gameplay mode.
- UI/UX.
- Game options.
- Development of the laser minigame.
- Dialog system.
Additionally, I modeled all the 3D assets myself using Blender.

This game was completed within a span of two weeks.
